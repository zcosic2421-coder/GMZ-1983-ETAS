# GMZ-1983-ETAS
Repozitorij GMZ-1983-ETAS sadrži dokumentaciju, el sheme, dijagrame i slike za uređaj Generator Morzeovih Znakova. GMZ je proizvod tvrtke Etas iz Splita u periodu 1983. do 1985. godine. Datum konstrukcije: 11. 1983. Konstruktor: Željko Ćosić dipl.ing.; Odobrio: Mladen Popović dipl. ing.; El. sheme Razradio: Sveto Lušić, Crtao: J. Ćuk
